# M9: Bridge Simplification and Manual Operating Surface

## Scope

Phase 9 defines a simplified daily operating model for Coshare.

It keeps existing advanced capabilities but moves them out of the default mental model.

The default question Bridge answers is:

```text
Can this task be accepted?
```

## Non-Goals

Do not implement:

- new platform integrations
- Multica API integration
- Multica API calls
- scheduler
- dashboard
- worker registry
- autonomous loop
- bidirectional sync
- automatic PR creation
- push automation
- auto-merge
- model API calls
- GitLab integration
- GitHub Projects integration
- new CLI commands
- new evidence types
- adapter code
- passing evidence solely to mark M9 accepted

## Evidence Pointer

Phase 9 is documentation-only unless a future task explicitly creates evidence.

If evidence is later produced, it must live under:

```text
docs/milestones/evidence/<task_id>/
```

## Closeout Pointer

Interim closeout lives at:

```text
docs/milestones/closeout/M9.md
```
