# M8: Real Multica Manual Trial

## Scope

Prepare and run a real manual Multica task trial.

The trial verifies whether a real Multica task can serve as the external execution substrate for Coshare while Bridge remains the final acceptance authority.

M8 prepares the trial only. It does not pretend the trial has already happened.

## Non-Goals

Do not implement:

- Multica API integration
- Multica API calls
- Bridge as a Multica client
- live reads inside Bridge gate
- Multica comments from Bridge
- Multica token or secret storage
- scheduler
- dashboard
- worker registry
- autonomous loop
- bidirectional sync
- automatic PR creation
- push automation
- auto-merge
- model API calls
- GitLab integration
- GitHub Projects integration
