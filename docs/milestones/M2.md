# M2: Git-backed Execution Baseline

## Scope

Build the Git-backed execution baseline for the agent workflow governance system.

M2 keeps milestone docs as the canonical task source and adds rules for git execution, role prompt packs, lane maps, and planned Bridge enhancements.

## Non-Goals

Do not implement:

- Multica integration
- GitHub Issue sync
- GitHub Issue as canonical task source
- GitLab integration
- dashboard
- auto-merge
- model API calls
- autonomous long-running loops
- worker metrics routing
- bidirectional sync
- production secret handling
- VPN/internal network automation

## Evidence Pointer

Structured evidence lives under:

```text
docs/milestones/evidence/<task_id>/
```

## Closeout Pointer

Closeout lives under:

```text
docs/milestones/closeout/
```
