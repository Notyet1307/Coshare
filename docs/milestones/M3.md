# M3: GitHub PR/CI Evidence Adapter

## Scope

Build the read-only GitHub PR/CI Evidence Adapter for the agent workflow governance system.

M3 keeps milestone docs as the canonical task source and adds normalized GitHub PR, CI/check, review, and branch protection evidence.

## Non-Goals

Do not implement:

- GitHub Issue as canonical task source
- GitHub Issue sync
- GitLab integration
- Multica integration
- dashboard
- auto-merge
- automatic PR creation
- push automation
- model API calls inside Bridge
- autonomous long-running loops
- worker metrics routing
- bidirectional sync
- production secret handling
- storing GitHub tokens in repo files

## Evidence Pointer

Structured evidence lives under:

```text
docs/milestones/evidence/<task_id>/
```

## Closeout Pointer

Closeout lives under:

```text
docs/milestones/closeout/
```
